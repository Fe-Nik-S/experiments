React. For beginners
=========


Installation
------------
Install project from git repository, by running:

```bash
    git clone https://github.com/Fe-Nik-S/experiments
    cd https://github.com/Fe-Nik-S/experiments
    make init_env
```


Usage
-----
For running service:

```bash
    make run_server
```

and open [http://localhost:3000](http://localhost:3000)
